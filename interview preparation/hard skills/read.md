Hard Skills (Technical Prep)

• Core coding (must-have)

  • Focus on:
    • arrays, objects
    • string manipulation
    • maps / sets
    • basic recursion
    • async logic (very important in Node.js)

  • Practice platforms:
    • LeetCode
    • HackerRank

  • Goal:
    • Write clean, correct, explainable code

Backend fundamentals (core strength)

  • APIs
    • REST principles
    • status codes:
      • 200, 201, 400, 401, 403, 500
    • request / response structure

  • Databases
    • SQL joins:
      • INNER JOIN

    • indexing (why it matters)
    • normalization basics
    • transactions

    • event loop (basic understanding)
