Soft Skills (THIS is where many fail)

• Structured thinking (critical)

  • Use this flow:
    • Clarify
    • Plan
    • Execute
    • Explain

  • Example behavior:
    • Ask clarifying questions first
    • Outline approach before coding
    • Then implement solution

Communication clarity

  • Avoid:
    • rambling
    • jumping between topics

  • Aim for:
    • short, structured explanations
    • “Here’s my approach…”

Tradeoff thinking

  • This is highly valued in interviews

  • Example:
    • “I’d use JWT here because it’s stateless,
      but sessions might be better if we need server-side control.”

Honesty under uncertainty

  • If you don’t know something:

    • Say:
      • “I’m not fully sure, but here’s how I’d approach it…”

  • Why this works:
    • Better than guessing blindly
    • Shows reasoning ability
