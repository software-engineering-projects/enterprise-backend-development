DESTRUCTURING (PROGRAMMING CONCEPT)

Definition
• Destructuring is a programming syntax used in languages such as JavaScript
• It allows unpacking values from arrays or properties from objects
• These values are assigned into separate variables in a single statement

---

CORE IDEA

• Instead of accessing values individually
• You extract multiple values at once
• You assign them directly into variables

---

ARRAY DESTRUCTURING

Purpose
• Extract values from arrays based on position

Example Concept
• Given an array of values
• Assign each value to a variable in order

Behavior
• Position matters
• Values are mapped sequentially

---

OBJECT DESTRUCTURING

Purpose
• Extract properties from objects by key

Example Concept
• Given an object with named properties
• Assign properties to variables with matching names

Behavior
• Key names matter
• Order does not matter

---

KEY CHARACTERISTICS

Conciseness
• Reduces multiple lines of variable assignment into one line

Readability
• Makes data extraction more explicit and structured

Flexibility
• Supports nested extraction in complex structures

---

INTENT

• Improve clarity when working with structured data
• Reduce repetitive code
• Enable cleaner handling of arrays and objects in application logic
